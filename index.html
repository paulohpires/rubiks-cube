Conhecendo o Cubo Magico

a) Centros

b) Meios

c) Quinas

Movimentos (Básicos)

a) R = Direita

b) L = Esquerda

c) U = Cima

Movimentos (intermediários)

a) R = Direita horário

b) RA = Direita anti-horário

c) R2 = Direita 2 vezes

Aprendendo os movimentos:

R + U + RA + UA = Nome do movimento = Direita

LA + UA + L + U = Nome do movimento = Esquerda
